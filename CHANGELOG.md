# Changelog

All notable changes to the AgentPsyAssessment project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- New multi-evaluator system for trustworthy assessments
- Segmented analysis methodology for improved reliability
- Batch processing capabilities for large-scale assessments
- Comprehensive result validation and consistency checking
- Web interface for simplified user interactions
- Advanced configuration management
- Support for multiple LLM providers (OpenAI, Anthropic, Ollama, Google)
- Integration with various psychometric models (Big Five, MBTI, cognitive functions)
- Detailed documentation and user manuals

### Changed
- Improved prompt engineering for more accurate psychological assessments
- Enhanced error handling and retry mechanisms
- Optimized performance for large-scale batch operations
- Updated API integration patterns for better reliability
- Refined personality role application for more nuanced assessments
- Better JSON parsing and error recovery mechanisms

### Fixed
- Resolved issues with inconsistent JSON outputs
- Fixed timeout handling for long-running assessments
- Corrected model configuration loading problems
- Addressed memory issues during large batch processing

- Basic CLI interface for running assessments

- Result storage and retrieval system
