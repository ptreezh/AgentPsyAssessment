# Portable PsyAgent - Claude Code Skills Specifications

## 项目概述

Portable PsyAgent 是一个便携式心理评估框架，结合了多种心理测量模型（大五人格、MBTI、贝尔宾团队角色等）与AI驱动的分析能力。本项目将这些核心功能转换为适用于 Claude Code 的标准化技能规范。

## 技能列表

### 1. psychological-assessment (心理评估技能)
**文件**: `psychological-assessment.md`

**功能概述**:
- 多理论模型支持（大五人格、MBTI、贝尔宾团队角色评估）
- 智能角色模拟（20种预定义人格角色）
- 压力测试分析（情绪压力、认知陷阱等）
- 多模型支持（OpenRouter、Ollama等）
- 实时评估和批量处理模式

**核心特性**:
- 支持50项大五人格完整评估
- 多级压力测试（0-4级）
- 认知陷阱分析（悖论、循环、语义、程序化）
- 质量控制和共识机制
- 综合报告生成

**使用场景**:
- 个人发展和自我认知
- 人才评估和团队配置
- 组织发展和文化建设
- 教育咨询和心理健康评估

## 技术架构特点

### 统一的设计原则
- **标准化输入输出**: 每个技能都有明确的JSON输入输出格式
- **模块化架构**: 技能可独立使用或组合使用
- **质量保证**: 内置可靠性验证和质量控制机制
- **隐私保护**: 符合伦理标准和数据隐私要求
- **可扩展性**: 支持自定义模型和扩展功能

### 核心技术组件
- **AI模型集成**: 支持多种AI模型和API
- **并发处理**: 高效的并发处理和资源管理
- **数据分析**: 深度的心理数据分析和挖掘
- **报告生成**: 专业的分析报告和建议生成
- **监控机制**: 实时的处理监控和质量保证

### 安全和伦理考虑
- **数据加密**: 全程数据加密和安全存储
- **隐私保护**: 严格的数据匿名化和访问控制
- **伦理指南**: 遵循心理评估的伦理标准
- **透明度**: 清晰的分析过程和结果解释

## 使用指南

### 基础使用流程
1. **技能选择**: 根据需求选择合适的技能
2. **数据准备**: 准备符合格式的输入数据
3. **配置参数**: 设置分析参数和选项
4. **执行分析**: 启动分析任务
5. **结果解读**: 获取和分析结果报告

### 高级使用场景
- **技能组合**: 多个技能组合使用获得全面分析
- **批量处理**: 大规模数据和群体分析
- **定制化**: 根据特定需求调整分析参数
- **集成应用**: 与现有系统和流程集成
