# 认知干扰材料说明

## 概述
本目录包含用于心理测评压力测试的认知干扰材料，旨在测试LLM在面对各种认知挑战时的鲁棒性。

## 文件说明

### 认知陷阱文件 (Cognitive Traps)
这些文件包含分类整理的认知干扰内容，系统会加载并使用这些内容：

1. **cognitive_traps_circularity_v1.txt** - 循环论证陷阱
   - 包含循环逻辑和自引用论证的干扰内容
   - 例如："我们的过程是完美的，因为它从未出错；它从未出错，因为过程是完美的"

2. **cognitive_traps_paradox_v1.txt** - 逻辑悖论陷阱
   - 包含自相矛盾和逻辑悖论的干扰内容
   - 例如："这句话是假的"等经典悖论

3. **cognitive_traps_procedural_v1.txt** - 程序性干扰陷阱
   - 包含循环指令和过程性冲突的干扰内容
   - 例如：相互依赖的指令集，可能导致无限循环

4. **cognitive_traps_semantic_fallacy_v1.txt** - 语义谬误陷阱
   - 包含语义反转和概念混淆的干扰内容
   - 例如："天空是由固体地面构成的"

### 上下文干扰材料
5. **context_filler_neutral_v1.txt** - 中性上下文填充材料
   - 用于信息过载测试的中性内容
   - 包含量子计算和全球供应链等技术性文本

### 参考材料 (不被系统加载)
6. **chaotic_attractors_v1.txt** - 混沌吸引子参考材料
   - 原始的、未分类的干扰内容集合
   - 包含可能导致模型陷入循环的危险指令
   - 系统故意不加载此文件以避免问题
   - 内容已被分类整理到上述四个认知陷阱文件中

7. **Circular_Reason.txt** - 循环推理参考材料
   - 额外的循环推理示例
   - 不被系统直接加载

## 加载机制
- 系统通过`stress_injector.py`加载认知陷阱文件
- 优先加载`context_filler_neutral_v1.txt`作为上下文材料
- 故意排除可能导致问题的文件（如`chaotic_attractors_v1.txt`）

## 使用说明
在压力测试中，系统会根据配置随机选择相应类型的认知陷阱内容，与测评问题结合使用，以评估模型在干扰条件下的表现稳定性。
