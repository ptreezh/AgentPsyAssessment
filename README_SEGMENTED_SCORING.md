# 独立评估器分段评分系统

## 项目简介
独立评估器分段评分系统是一个用于对大型心理测评报告进行可信评估分析的工具。该系统通过将大型测评报告分割成小段（每段5题），使用多个模型进行独立评估，并通过一致性检验和信度验证确保评估结果的可靠性。

## 核心功能

### 1. 分段评分
- 将大型测评报告按5题一段进行分割
- 每段使用独立的评估流程，减少上下文长度压力
- 支持大规模报告的处理（最多50题）

### 2. 多模型评估
- 集成多个云模型（OpenRouter API）
- 支持本地模型（Ollama）
- 自动故障转移和多级重试机制

### 3. 一致性检验
- 计算模型间的一致性指标
- 识别评分分歧和异常值
- 提供详细的分歧分析报告

### 4. 信度验证
- 计算Cronbach's Alpha系数
- 评估者间信度分析
- 设置信度阈值验证（默认0.8）

### 5. 分歧处理
- 多轮争议解决机制（最多3轮）
- 动态增加评估器
- 多数一致判定原则

## 系统架构

### 模块化设计
- `SegmentedScoringEvaluator`: 核心评估器
- `ReliabilityValidator`: 信度验证器
- `DisputeResolutionManager`: 分歧处理管理器
- `APIClient`: 多API客户端

### 多级重试机制
1. 首选云模型（OpenRouter API）
2. 备选本地模型（Ollama）
3. 指数退避重试策略
4. 最多3次重试

### 安全性
- API密钥通过环境变量管理
- 不在源代码中硬编码敏感信息
- 实施API密钥安全管理最佳实践

## 使用场景
- 心理测评报告的可信评估分析
- 大型评估数据的批量处理
- 多模型评估结果的一致性检验
- 评估结果的信度验证
