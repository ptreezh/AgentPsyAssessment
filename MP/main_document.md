# AgentPsy 产品说明书

## 1. 项目概述

AgentPsy 是一个基于大语言模型（LLM）的心理测评系统，旨在通过自动化的方式对AI代理进行心理特征评估。该系统支持多种心理测评量表，包括大五人格（Big Five）、抑郁焦虑压力量表（DASS）、动机测试等，并提供了丰富的压力测试功能来评估模型在不同情境下的表现。

## 2. 核心功能

- **多模型支持**：支持本地模型（如Ollama）和云模型（如GLM、DeepSeek等）
- **多样化测评**：支持多种心理测评量表
- **压力测试**：提供情感压力、认知陷阱、上下文负载等多种压力测试功能
- **角色扮演**：支持为模型分配不同角色进行测评
- **结果分析**：提供详细的测评结果分析和报告生成

## 3. 技术亮点

- **模块化架构**：采用模块化设计，便于扩展和维护
- **统一接口**：提供统一的模型服务接口，简化模型管理
- **压力注入**：支持多种压力因子的动态注入
- **会话隔离**：确保每个测评问题的独立性和隔离性
- **日志记录**：完整的测评过程日志记录和错误处理

## 4. 测评方法与条件

- **测评流程**：通过`run_assessment_unified.py`脚本启动测评，支持多种参数配置
- **压力因子**：包括情感压力等级（0-4）、认知陷阱类型（悖论、循环性、语义谬误、程序性）、上下文负载等
- **模型配置**：支持温度参数、上下文长度等模型参数配置

## 5. 应用场景与展望

- **AI代理评估**：评估AI代理在不同角色和压力下的心理特征表现
- **模型对比**：对比不同模型在相同测评任务下的表现差异
- **持续优化**：通过测评结果反馈，持续优化模型性能和行为
