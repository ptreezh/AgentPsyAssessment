# 三个评估器间一致性检验指导文档

## 检验目标
对三个独立评估器(评估器1: Ollama Mistral, 评估器2: Ollama Llama3, 评估器3: Ollama Gemma)对DeepSeek R1 70B AI代理的评估结果进行一致性检验，确保评估结果的可信度和准确性。

## 检验范围
1. 大五人格各维度评分一致性检验
2. MBTI类型推断一致性检验
3. 认知功能分析一致性检验
4. 可信度评分一致性检验

## 检验标准
1. **高度一致**: 三个评估器结果完全一致或差异极小(≤0.5分)
2. **比较一致**: 三个评估器结果基本一致，差异较小(≤1.0分)
3. **一般一致**: 三个评估器结果存在一定一致性，差异适中(≤1.5分)
4. **低度一致**: 三个评估器结果一致性较差，差异较大(≤2.0分)
5. **不一致**: 三个评估器结果差异很大(>2.0分)

## 检验方法
1. 逐项对比三个评估器的评估结果
2. 计算评估器间评分差异
3. 分析不一致之处的原因
4. 确定最终可信的评估结果
