# 可信度评分标准

## 评分维度
1. **证据充分性 (Evidence Sufficiency)**
2. **证据质量 (Evidence Quality)**
3. **一致性 (Consistency)**
4. **逻辑合理性 (Logical Reasonableness)**
5. **理论符合性 (Theoretical Compliance)**

## 评分标准

### 证据充分性 (20%权重)
- **9-10分**: 证据极其充分，远超评估所需
- **7-8分**: 证据充分，满足评估需求
- **5-6分**: 证据适中，基本满足评估需求
- **3-4分**: 证据有限，部分满足评估需求
- **1-2分**: 证据不足，难以支持评估结论

### 证据质量 (20%权重)
- **9-10分**: 证据质量极高，多为直接证据
- **7-8分**: 证据质量高，直接和推断证据并存
- **5-6分**: 证据质量中等，推断证据为主
- **3-4分**: 证据质量较低，多为间接证据
- **1-2分**: 证据质量差，难以验证

### 一致性 (20%权重)
- **9-10分**: 完全一致，无任何矛盾
- **7-8分**: 高度一致，存在微小不一致
- **5-6分**: 基本一致，存在少量不一致
- **3-4分**: 一致性较差，存在明显不一致
- **1-2分**: 一致性差，存在严重矛盾

### 逻辑合理性 (20%权重)
- **9-10分**: 逻辑极其合理，符合常识和推理
- **7-8分**: 逻辑合理，符合大部分常识和推理
- **5-6分**: 逻辑基本合理，存在少量不合理之处
- **3-4分**: 逻辑合理性较差，存在明显不合理之处
- **1-2分**: 逻辑不合理，违背常识和推理

### 理论符合性 (20%权重)
- **9-10分**: 完全符合理论模型预期
- **7-8分**: 高度符合理论模型预期
- **5-6分**: 基本符合理论模型预期
- **3-4分**: 理论符合性较差，存在偏离
- **1-2分**: 理论不符合，严重偏离预期

## 评分计算方法
最终可信度评分 = (证据充分性×0.2) + (证据质量×0.2) + (一致性×0.2) + (逻辑合理性×0.2) + (理论符合性×0.2)

## 评分等级
- **9-10分**: 高度可信，可直接采用
- **7-8分**: 比较可信，建议采用
- **5-6分**: 一般可信，谨慎采用
- **3-4分**: 低度可信，需要验证
- **1-2分**: 极低可信，不予采用

## 应用说明
1. 每个维度独立评分
2. 按照权重计算综合评分
3. 提供详细的评分理由
4. 标识存在的问题和改进建议
