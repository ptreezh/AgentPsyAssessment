# 证据审查指导文档

## 审查目标
对所有用于大五人格和MBTI类型评估的证据进行系统性审查，确保评估结果的可信度和准确性。

## 审查范围
1. 大五人格各维度证据审查
2. MBTI类型推断证据审查
3. 认知功能分析证据审查
4. 一致性检验证据审查

## 审查标准
1. **证据相关性**: 证据必须与所评估的维度直接相关
2. **证据充分性**: 每个维度必须有足够数量的证据支持
3. **证据质量**: 证据必须具有明确的来源和可验证性
4. **证据一致性**: 不同证据之间不应存在明显矛盾

## 审查方法
1. 逐项检查每个维度的证据
2. 评估证据的相关性、充分性和质量
3. 检查证据之间的一致性
4. 标识任何可疑或不一致的证据

## 证据分类
### 1. 直接证据
- 来自测评报告中明确提及的特征
- 具有明确来源和引用

### 2. 推断证据
- 基于测评报告内容的合理推断
- 具有逻辑基础和可验证性

### 3. 间接证据
- 通过行为模式或回应方式间接体现的特征
- 需要更多的解释和论证

## 审查报告模板

### 证据审查报告结构:
1. **审查目标**: 明确审查的维度和目标
2. **证据清单**: 列出所有用于该维度评估的证据
3. **审查结果**: 
   - 相关性评估
   - 充分性评估
   - 质量评估
   - 一致性评估
4. **审查结论**: 综合评估结果
5. **改进建议**: 如有不足，提出改进建议

### 一致性检验报告结构:
1. **检验目标**: 明确检验的维度和目标
2. **检验范围**: 列出需要检验的维度组合
3. **检验结果**: 
   - 大五人格各维度间一致性
   - MBTI类型与大五人格一致性
   - 认知功能与MBTI类型一致性
4. **检验结论**: 综合检验结果
5. **问题标识**: 标识任何不一致之处

## 质量控制
1. **独立审查**: 每个维度独立审查，避免相互影响
2. **证据导向**: 所有审查结论必须基于具体证据
3. **一致性检验**: 确保各维度间逻辑一致
4. **透明记录**: 所有审查过程和结论必须明确记录
