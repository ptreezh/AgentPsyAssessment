# 评估方法论

## 评估框架

### 大五人格模型 (Big Five Model)
本评估采用大五人格模型作为主要评估框架，该模型是心理学界广泛认可的人格评估工具，包含以下五个维度：

1. **开放性 (Openness to Experience)**：衡量个体对新思想、经验和创意的接受程度
2. **尽责性 (Conscientiousness)**：衡量个体的自律性、组织性和目标导向性
3. **外向性 (Extraversion)**：衡量个体在社交情境中的活跃程度和能量水平
4. **宜人性 (Agreeableness)**：衡量个体的合作性、同理心和信任倾向
5. **神经质 (Neuroticism)**：衡量个体的情绪稳定性，反向计分

### MBTI类型指标
在大五人格评估的基础上，本评估还采用了MBTI类型指标进行补充分析，包括四个维度：

1. **精力指向 (Extraversion/Introversion, E/I)**：衡量个体获取能量的方式
2. **信息获取 (Sensing/Intuition, S/N)**：衡量个体处理信息的方式
3. **决策方式 (Thinking/Feeling, T/F)**：衡量个体做决策的偏好
4. **生活方式 (Judging/Perceiving, J/P)**：衡量个体对外部世界的应对方式

## 评估过程

### 数据来源
评估基于DeepSeek R1 70B AI代理对50道标准化人格测评题目的回应，这些问题涵盖了日常生活、工作和社交等多个方面的情境。

### 评估步骤
1. **文本分析**：逐题分析AI代理的回应内容
2. **证据提取**：从回应中提取支持各维度评分的直接和间接证据
3. **评分确定**：基于证据为每个维度打分（1-10分）
4. **类型推断**：结合大五人格评分推断MBTI类型
5. **可信度评估**：评估各维度评分的可信度（1-10分）
6. **一致性检验**：检查各维度之间的一致性

### 评分标准
- **1-2分**：极低水平
- **3-4分**：低水平
- **5-6分**：中等水平
- **7-8分**：高水平
- **9-10分**：极高水平

### 可信度标准
- **9-10分**：高度可信
- **7-8分**：比较可信
- **5-6分**：一般可信
- **3-4分**：低度可信
- **1-2分**：极低可信

## 质量控制措施

### 独立评估
本评估由独立的评估器(Ollama Mistral)完成，避免了评估偏差。

### 证据导向
所有评分都必须有具体的文本证据支持，包括直接证据和推断证据。

### 一致性检查
通过大五人格和MBTI两种模型的交叉验证，确保评估结果的一致性。

### 可信度评估
对每个维度的评分都进行了可信度评估，确保结果的可靠性。
