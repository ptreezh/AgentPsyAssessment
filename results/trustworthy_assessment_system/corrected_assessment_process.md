# 可信评估流程 - 严格按照您的要求实施

## 正确的评估流程

### 第一阶段：为每个原始测评报告创建三个独立评估器的评分环境
1. 为每个测评报告创建独立的评估目录
2. 为每个评估器创建独立的工作空间
3. 复制原始测评报告到各个评估器工作目录

### 第二阶段：实施独立评估器对每道题进行评分
1. 每个评估器独立对原始测评报告的50道题目逐题评分
2. 每道题评分范围：1-10分
3. 每道题必须提供评分依据（直接证据/推断证据）
4. 每个评估器生成独立的题目评分文件

### 第三阶段：进行评估器间一致性检验
1. 对比三个评估器对每道题的评分
2. 计算每道题的评分一致性系数
3. 识别一致性低于阈值的题目（例如差异超过2分）

### 第四阶段：对分歧大的题目增加额外评估器评分
1. 对一致性不足的题目增加两个额外评估器（总共5个评估器）
2. 重新对这些题目进行评分
3. 去掉最高分和最低分后重新计算一致性

### 第五阶段：形成最终可信的题目评分
1. 对一致性达标的题目采用平均分作为最终评分
2. 对经过5评估器评分后仍一致的题目采用去掉极值后的平均分
3. 标记仍存在分歧的题目并说明原因

### 第六阶段：基于可信题目评分进行大五人格和MBTI分析
1. 基于可信的题目评分计算大五人格各维度得分
2. 基于大五人格得分推断MBTI类型
3. 进行认知功能分析

## 关键原则
1. **题目级评分**: 每个评估器对每道题独立评分
2. **一致性检验**: 基于题目级别的评分一致性
3. **分歧处理**: 对分歧大的题目增加评估器重新评分
4. **可信度验证**: 只有通过一致性检验的评分才认为可信
5. **后续分析**: 只有在题目评分可信后才进行大五人格和MBTI分析

## 实施步骤

### 评估器配置
- 评估器1: Ollama Mistral
- 评估器2: Ollama Llama3
- 评估器3: Ollama Gemma
- 额外评估器4: Ollama Phi3
- 额外评估器5: Ollama Qwen2

### 一致性阈值
- 高一致性: 评分差异≤1分
- 中等一致性: 评分差异2分
- 低一致性: 评分差异≥3分

### 处理流程
1. 三个评估器评分 → 一致性检验 → 
   - 高一致性题目: 直接采用平均分
   - 中等一致性题目: 增加两个评估器重新评分
   - 低一致性题目: 增加两个评估器重新评分
2. 五个评估器评分 → 去掉极值 → 重新一致性检验 →
   - 一致性达标: 采用去掉极值后的平均分
   - 一致性仍不足: 标记为不可信并说明原因
